Website Creation Project (Automatic Question Generation Platform)

July 2024 - December 2024

Proyek ini adalah pengembangan sistem Automatic Question Generation (AQG) berbasis ringkasan yang dirancang untuk membantu pengajar dalam membuat soal secara otomatis. Sistem ini dibangun menggunakan Django sebagai framework backend, dengan Python sebagai bahasa pemrograman utama, serta HTML, CSS, dan JavaScript untuk tampilan frontend. Website ini memungkinkan pengguna untuk menghasilkan soal dari ringkasan teks secara otomatis, dengan berbagai fitur pendukung untuk memastikan kualitas soal yang dihasilkan.

Fitur dan Tampilan:

  • Slide 1: Menampilkan halaman awal dari website AQG Summary.
  • Slide 2: Menunjukkan hasil ringkasan buku yang telah diproses sebelumnya menggunakan teknik ekstraksi dan abstraksi.
  • Slide 3: Memperlihatkan fitur evaluasi akurasi ringkasan, yang digunakan untuk mengukur kualitas ringkasan sebelum digunakan dalam pembuatan soal.
  • Slide 4: Menampilkan fitur utama sistem, yaitu pembuatan soal otomatis dengan opsi untuk memilih tipe soal serta jumlah soal hingga maksimal 10 soal.
  • Slide 5: Menunjukkan hasil akhir dari seluruh proses, termasuk tampilan ringkasan, soal yang dihasilkan, serta akurasi dari ringkasan tersebut.

Data Analyst Project (Cancer Patients)

October 2023 - December 2023

Proyek ini dilakukan dalam mata kuliah Machine Learning, di mana saya melakukan analisis data pasien kanker menggunakan tiga algoritma klasifikasi: Random Forest, XGBoost, dan Support Vector Machine (SVM) untuk membandingkan performa masing-masing dalam mengklasifikasikan jenis kanker berdasarkan data teks yang berisi gejala pasien. Dataset yang digunakan berasal dari Kaggle dengan 7.569 baris data, terdiri dari dua kolom utama: Text (deskripsi gejala pasien) dan Target (klasifikasi kanker) yang mencakup tiga jenis kanker, yaitu Colon Cancer (Kanker Usus Besar), Lung Cancer (Kanker Paru-Paru), dan Thyroid Cancer (Kanker Tiroid). Metode yang digunakan dalam analisis ini meliputi preprocessing data dengan penghapusan karakter khusus, konversi teks ke huruf kecil, stopword removal, dan penghapusan kata pendek, serta pembagian dataset menjadi 80% data latih dan 20% data uji untuk meningkatkan akurasi model.

Tampilan:

  • Slide 1: Dataset mentah sebelum dilakukan preprocessing.
  • Slide 2-4: Proses preprocessing data untuk meningkatkan kualitas input model.
  • Slide 5-7: Hasil klasifikasi menggunakan Random Forest.
  • Slide 8-10: Hasil klasifikasi menggunakan XGBoost.
  • Slide 11-13: Hasil klasifikasi menggunakan SVM.

Data Analyst Project (Airline Delay)

September 2023 - October 2023

Dalam mata kuliah Machine Learning, saya mengembangkan sebuah proyek analisis prediksi keterlambatan penerbangan dengan menggunakan tiga algoritma pembelajaran mesin, yaitu K-Nearest Neighbor (KNN), Naïve Bayes, dan Random Forest. Proyek ini bertujuan untuk membandingkan efektivitas berbagai metode klasifikasi dalam memprediksi keterlambatan penerbangan maskapai. Data diperoleh dari Kaggle dengan jumlah 539.382 observasi dan terdiri dari 6 atribut utama yang digunakan sebagai fitur. Dataset dibagi menjadi data latih (60%) dan data uji (40%) untuk evaluasi model.

Tampilan:

  • Slide 1: Dataset yang digunakan dalam analisis.
  • Slide 2: Hasil klasifikasi menggunakan algoritma K-Nearest Neighbor.
  • Slide 3: Hasil klasifikasi menggunakan algoritma Naïve Bayes.
  • Slide 4: Hasil klasifikasi menggunakan algoritma Random Forest.

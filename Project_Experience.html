Data Analyst Project (Cancer Patients)

October 2023 - December 2023

This project was done in the Machine Learning course, where I analyzed cancer patient data using three classification algorithms: Random Forest, XGBoost, and Support Vector Machine (SVM) to compare their performance in classifying cancer types based on text data containing patient symptoms. The dataset used comes from Kaggle with 7,569 rows of data, consisting of two main columns: Text (description of patient symptoms) and Target (cancer classification) which includes three types of cancer, namely Colon Cancer, Lung Cancer, and Thyroid Cancer. The methods used in this analysis include data preprocessing with special character removal, text conversion to lowercase, stopword removal, and short word removal, as well as dividing the dataset into 80% training data and 20% test data to improve model accuracy.

Views:

  • Slide 1: The raw dataset before preprocessing.
  • Slide 2-4: Data preprocessing to improve the quality of model inputs.
  • Slide 5-7: Classification results using Random Forest.
  • Slide 8-10: Classification results using XGBoost.
  • Slide 11-13: Classification results using SVM.
